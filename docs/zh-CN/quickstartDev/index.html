快速入门

LPWAN 应用服务器

环境

设置 Docker

  • 安装Docker 只需跟随 使用资源库/设置资源库无需安装Docker引擎社区

  • 安装 docker-compose 只需跟随 在 Linux 系中安装 Compose

  • 添加用户到docker组

sudo usermod -aG docker $USER

克隆资源库并获取最新的开发分支：

git clone https://github.com/mxc-foundation/lpwan-app-server.git
cd lpwan-app-server
git checkout develop
git pull

使用本地数据库开始容器并构建服务&mqtt

  • 在容器外面，执行
$LPWAN-APP-SERVER-PATH/appserver local

构建后，服务将以配置文件./configuration/lora-app-server.toml开始运行。

  • 当你输入 Ctrl+c 停止使用服务时，你将在容器内；修改代码后，如果你想要重建，请停留在容器内，然后执行:
$LPWAN-APP-SERVER-PATH/appserver build

构建完成后，服务将以配置文件./configuration/lora-app-server.toml开始运行。

您可以在浏览器中访问 http://localhost:8080 预览服务。

提示： 如果你在构建界面时遇到任何困难，尝试在容器内： bash cd $LPWAN-APP-SERVER-PATH/ui rm package-lock.json npm install cd .. ./appserver build

MXC极域协议服务器

克隆资源库并获取最新的开发分支：

git clone https://github.com/mxc-foundation/mxprotocol-server.git
cd mxprotocol-server
git checkout develop
git pull

使用本地数据库启用容器并构建服务

  • 在容器外面，执行
$MXPROTOCOL-SERVER-PATH/mxprotocol-server local_database

构建完成后，服务器将开始使用配置文件运行$MXPROTOCOL-SERVER-PATH/configuration/mxprotocol-server.toml.

  • 当你输入 Ctrl+c 停止使用服务时，你将在容器内；修改代码后，如果你想要重建，请停留在容器内，然后执行:
$MXPROTOCOL-SERVER-PATH/mxprotocol-server build

构建完成后，服务器将开始使用配置文件运行$MXPROTOCOL-SERVER-PATH/configuration/mxprotocol-server.toml.

提示： 如果您想要连接应用服务器和MXC极域协议服务器，您需要在容器外运行此 __

docker ps
docker network ls

您可以在输出中找到网络名称和容器 ID，然后运行它

docker network connect NETWORK_NAME_APPSERVER MXPROTOCOL_CONTAINER_ID
docker network connect NETWORK_NAME_MXPROTOCOL APPSERVER_CONTAINER_ID
