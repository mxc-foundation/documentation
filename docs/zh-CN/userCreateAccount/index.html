我如何创建一个账户？

要创建 MXC极域超级节点账户，您首先需要导航到MXC极域支持的超级节点选项。 您可以在我们的 开始使用 页面中找到这个列表。 一旦您处于您选择的超级节点，请遵循这些说明。

提交您的电子邮件地址

  1. 点击 "注册"
  2. 请输入您的电子邮件地址
  3. 完成验证码
  4. 点击 "注册"

创建密码

在您输入您的电子邮件地址后，您将收到超级节点的电子邮件。 这可能会落在您的垃圾邮件文件夹中，所以如果您没有收到电子邮件，请检查垃圾邮件

  1. 打开电子邮件
  2. 点击邮件中的链接
  3. 输入您的密码

登录

现在您可以使用您的电子邮件地址和密码登录到您的超级节点帐户。
